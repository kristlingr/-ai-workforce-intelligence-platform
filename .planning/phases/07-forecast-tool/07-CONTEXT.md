# Phase 7 Context: Forecast Tool & Forecast Agent

## Purpose
Implement the prediction core area of the Workforce Intelligence System:
1. `ForecastTool`: Calculates department capacity, staffing allocations demand, and resource gaps in hours and FTE.
2. `ForecastAgent`: AI Agent invoking the `ForecastTool` and using Gemini LLM to synthesize narrative capacity reports.

## Requirements
- **TOOL-04**: ForecastTool rolling capacity and staffing demand calculations.
- **AGENT-04**: ForecastAgent capacity and shortage reporting.
