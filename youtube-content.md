## Title

AI Workforce Intelligence Platform — Multi-Agent Demo

## Description

An autonomous multi-agent system that transforms workforce data into executive-ready strategic decisions. Type a natural language question like "Show me utilization risks in Engineering" and 7 specialized AI agents analyze data, detect risks, forecast capacity, and deliver a structured report — all in under 10 seconds.

Built with:
- Python / Streamlit
- Google Gemini + OpenAI (fallback)
- Multi-Agent Orchestration (Manager Agent → 4 sub-agents)
- Deterministic Tool Layer (EmployeeLookup, ForecastTool, etc.)
- Executive Report Engine with Validation & Evidence Traceability

Live demo: https://kristlingr--ai-workforce-intelligence-platform-app-zzr6ww.streamlit.app/
GitHub: https://github.com/kristlingr/-ai-workforce-intelligence-platform
