Proud to present my capstone project for the Kaggle x Google Vibe Coding Agentic AI Challenge 🚀

I built AI Workforce Intelligence Platform — an autonomous multi-agent system where you type "Where should we hire next quarter?" and 7 specialized AI agents analyze workforce data, forecast capacity, and deliver executive-ready recommendations in seconds.

The core insight: tools for facts, LLM for language. Data queries go through deterministic tools — the LLM never touches raw data. It only synthesizes results into narrative. No hallucination, full evidence traceability.

Stack: Python, Streamlit, Google Gemini, with Manager Agent orchestrating 4 sub-agents (Workforce, Utilization, Forecast, Recommendation).
